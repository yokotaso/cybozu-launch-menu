# cybozu-launch-menu
このプロジェクトは、サイボウズ社内で利用されている
お弁当屋さん たまごや・嵯峨野の日替わり弁当メニューを
取得するCGIスクリプトです

# 実行の仕方

```bash
ruby menu.rb
```
* menu.rbはCGIスクリプトになっています。

# 利用言語およびライブラリ 
* ruby 1.8.7
* nokogiri (1.4.4)
* rspec(2.14.1)
* mocha(0.14.0)
